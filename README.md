# 帕鲁伙伴编辑器

面向 Steam 版 Palworld 1.0 的本地中文伙伴存档编辑器。

## 下载

无需 Python 的 Windows 独立版位于 `release/帕鲁伙伴编辑器_v1.3.0.zip`。
源码构建方法见 `BUILD.md`。

## 功能

- 自动定位最近使用的 `Level.sav`
- 按 Steam 玩家列出背包和终端中的伙伴
- 修改生命 IV、近战/远程攻击 IV
- 修改生命、攻击、防御 IV/灵魂强化以及工作速度强化
- 超限模式支持 IV/灵魂强化 0～255、浓缩等级 0～254
- 可将当前玩家的全部伙伴一键设为超限战斗最大值，并将各自已有工作设为 10
- 一键补齐当前玩家尚未拥有的全部正常可获得物种/亚种，每种一只并放入帕鲁终端
- 使用 Palworld 1.0 数据表显示 299 个物种的简体中文名称
- 只显示该物种实际拥有的工作适应性，并可设置最终等级 1～10
- 识别异常浓缩 `Rank`，不再把 127 等原始值误显示为 126 星
- 保存前检查游戏进程和存档外部变更
- 自动备份、临时写入、回读验证、原子替换
- 新增伙伴时同时验证实体、终端槽位和公会成员记录；1.0 新增的公会尾部数据原样保留

## 安全提示

必须先正常退出 Palworld 再保存。备份保存在世界存档目录的
`PalPartnerEditor_Backups` 文件夹中。

本项目复用了 PalEdit 的 MIT 许可数据解析代码。许可证见
`THIRD_PARTY_PALEDIT_LICENSE.txt`。

Palworld 1.0 物种名称和工作适应性数据来自 Pal Calc v1.17.7（数据库 v26），
其 MIT 许可证见 `THIRD_PARTY_PALCALC_LICENSE.txt`。
